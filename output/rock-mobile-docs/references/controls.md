# Controls

Views are user-interface objects such as labels, buttons, and sliders that are commonly known as controls or widgets in other graphical programming environments.

**.NET MAUI Base Views**

The .NET MAUI framework provides a rich collection of base views (controls). You can find a documentation on these views on the [Microsoft site](https://learn.microsoft.com/en-us/dotnet/maui/user-interface/controls/?view=net-maui-8.0).
