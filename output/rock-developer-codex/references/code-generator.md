# Code Generator

Because there are a number of tasks that are repetitive, we have a code generator tool that will automate many things related to models. There are also other generation options available in the tool.

![](https://community.rockrms.com/GetImage.ashx?Id=66688)

Warning: The Code Generation tool uses the DLLs that are part of the main solution. This means you need to make sure you compile the Rock solution before trying to run the Code Generation tool, otherwise it might try to use old data.
