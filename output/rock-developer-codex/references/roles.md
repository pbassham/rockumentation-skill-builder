# Roles

To ensure that we have good coverage of different environments the following people will take on the roles below.

-   Run Edge (~Mike Peterson~) – Will run Edge (aka Edgium) as their primary browser when running and debugging Rock.
-   Snap-shot Isolation Off (~Mike Peterson~) – Will ensure all of his local databases have Snapshot Isolation Off.
-   Run IISExpress in 32 bit mode (*Someone other than Mike*) – This is to ensure we are not running into memory issues.
-   Run using the oldest supported version of SQL Server (Nick) -- This ensures we don't accidentally use a SQL command that SQL Server doesn't yet support.
