# Spark/Organization Commits

*Commits to the Spark repository should be in the format:*

***Client*: *Project* - *Commit Note***

For example, if I was committing a change to work on a project for a new prayer wall for Rock Solid Church an example commit might look like this:

“Rock Solid Church: Prayer Wall - Added Prayer Wall block”
