# Daily Commits \[-\]

These commits represent changes/additions that are not important enough for release notes:

-   Must start with a `–`
    -   `- Added Validation Alert to Note Type detail block`.
-   Should not auto-close any issues (these should be release commits).

Caution

Do not include issue numbers (#0000) in a commit unless you have fixed the referenced issue and it is your FINAL commit.
