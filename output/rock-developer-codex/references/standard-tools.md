# Standard Tools

It’s important that we all use the standard tools in the development of Rock. Some flexibility can be had with fringe tools that are used outside of the scope of the tools below. These tools though should be approved by the Product Experience Director (PED) before use.

-   IDE – Visual Studio (version defined by PED) – This should be used for all source C#, HTML, Less, CSS, Javascript, etc.
-   Unit Test platform is MSTest.
-   SQL Editor – SQL Server Management Studio
-   Compare Tool – Beyond Compare ($) or WinMerge (free)
-   Git Client – SmartGit
-   Less Compiler – Rock or ~WinLess~
-   Mockups – Figma
-   Office – Microsoft Office
-   Image Editing – Photoshop / Illustrator
