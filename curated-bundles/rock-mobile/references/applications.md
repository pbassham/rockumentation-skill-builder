## Application Settings

When creating or editing a Roku application, you have access to the following configuration options.

![Application Configuration Options](https://community.rockrms.com/GetImage.ashx?Id=66728)

### Enable Page Views

Whether (or not) page interactions should be written to track the usage of your application.

### Page View Retention Duration

The duration (in days) to retain the page interactions that are written.  

### API Key

The API key used to securely connect to your Roku application.

### Authentication Page

The **website** authentication page that can be used to remotely authenticate in your TV application. Used
